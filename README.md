# NestSong
PRN231 Project UI Using React + Vite 

### Installation
### Clone repo
``` bash
# clone the repo
git clone https://github.com/PMNhutt/NestSong.git

# go into app's directory
cd NestSong

# install app's dependencies
yarn install
``` 

### Usage
``` bash
# serve with hot reload at http://127.0.0.1:517/
yarn dev
```
